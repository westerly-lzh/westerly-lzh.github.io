Security Solution of IT Environment

Security Controls

The Security Controls are a prioritized set of Safeguards to mitigate the most prevalent cyber-attacks against systems and networks. They are mapped to and referenced by multiple legal, regulatory, and policy frameworks. Security Controls has been enhanced to keep up with modern systems and software. Movement to cloud-based computing, virtualization, mobility, outsourcing, Work-from-Home, and changing attacker tactics prompted the update and supports an enterpriseâ€™s security as they move to both fully cloud and hybrid environments.

Security Controls are now support Implementation Groups (IGs), a simple and accessible way to help organizations of different classes focus their scarce security resources. IGs are the recommended guidance to prioritize implementation of the Security Controls. In an effort to assist enterprises of every size, IGs are divided into three groups. They are based on the risk profile and resources an enterprise has available to them to implement the Security Controls. Each IG identifies a set of Safeguards, that they need to implement.

CIS Security Benchmarks

Security Benchmarks are best practices for the secure configuration of a target system. Available for more than 100 Benchmarks across 25+ vendor product families, Security Benchmarks are developed through a unique consensus-based process comprised of cybersecurity professionals and subject matter experts around the world. Security Benchmarks are the only consensus-based, best-practice security configuration guides both developed and accepted by government, business, industry, and academia.
